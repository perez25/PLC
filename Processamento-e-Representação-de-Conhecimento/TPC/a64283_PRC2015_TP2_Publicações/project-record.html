Abstract

Motivação:

A Web Semântica é nada mais nada menos, que uma web com toda sua informação organizada de forma que não somente seres humanos possam entendê-la, mas principalmente máquinas/computadores. Disse principalmente máquinas, porque elas nos ajudarão, de fato, em tarefas que hoje, invariavelmente temos que executá-las manualmente.

Objectivo :

Terminar as ocorrências do ficheiro: jcrpubs.xml

Para tal foi necessário analisar o ficheiro , de modo a ver quais eram as propriedades de cada tipo de Publicação. Foi possível concluir que existem as seguintes subclasses de Publicação :

  • Book
  • Proceedings
  • Inproceedings
  • Phdthesis
  • MasterThesis
  • Article
  • Misc
  • Inbook

    • Através da análise de cada tipo foi possível inferir que todos tinham as propriedades:

  • Título
  • Doi
  • Data
  • Autor

    • Contudo , também foi possível inferir que existem propriedades que são destinadas somente a cada tipo de publicação. Por exemplo tanto Phdthesis, como Masterthesis contêm a propriedade school.

    Por outro lado o tipo Article é o que contêm mais propriedades unicas, tais como:

  • number
  • issn
  • volume
  • journal

    • Para representar estes dados foi usado XSL de modo a processar o ficheiro jcrpubs.xml com a finalidade de originar um ficheiro com os dados representados na linguagem RDF
