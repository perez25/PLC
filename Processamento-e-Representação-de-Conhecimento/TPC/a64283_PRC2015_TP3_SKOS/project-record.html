Abstract

Motivação:

O Simple Knowledge OrganizationSystem (SKOS) é um modelo para expressar a estrutura básica e conteúdo de sistemas de organização do conhecimento. SKOS é, como o nome representa, um sistema simples para a organização do conhecimento.

Objectivo :

Analisar e processar o ficheiro ACM-SKOSTaxonomy.xml com a finalidade, de criar um indice com os top-concepts e com os concepts.

Para além da criação do indice,foi criado para cada concept, uma página com o título , etiquetas/classificadores e com as suas relações semânticas.

Os dados estavam modelados em SKOS. Estes referenciavam uma data de conceitos, onde, cada um destes estava identificado por um id. Através deste foi possível interligar os diferentes conceitos de modo a construir um multi grafo. De modo a construir este foi necessário estudar e analisar as relações semânticas de cada conceito , nomeadamente os seus broaders,narrowers e os conceitos que estão relacionados related. No ficheiro ACM-SKOSTaxonomy.xml estas relações vinham acompanhadas com uma propriedade resource que continha o id para os conceitos.

Os conceitos foram organizados por ordem alfabética e para apresentar os dados foi utilizado bootstrap.
