Abstract

Para a elaboração deste TPC era necessário requer conhecimentos sobre XSL. Este descreve como fazer o “display” de um documento XML. XSL é uma espécie stylesheets para xml, do mesmo modo que CSS são stylesheets para HTML. O XSL engloba três partes :

XPATH - linguagem para definir partes de um documentos XML;

XSLT - linguagem para tranformar documentos XML;

XSL-FO - linguagem para definir documentos XML;

Objectivo

Criar uma stylesheet que processe informação sobre a submissão de trabalhos práticos, e criar uma página HTML onde seja fácil consultar a informação das várias submissões. Por fim completar o project-record.
